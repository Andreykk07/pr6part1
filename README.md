# Backend Developer Prompt Library

Пакет параметризованих та структурованих промптів для автоматизації щоденних завдань backend-розробника (Express.js, Prisma, Typecript, Jest, Docker).

## Огляд бібліотеки
Бібліотека реалізована як TypeScript модуль та містить 10 шаблонів функцій, які динамічно приймають параметри вашого контексту та повертають оптимізований промпт для LLM (ChatGPT, Claude).

### Використані техніки промптингу:
* **Zero-shot** (Прямі запити для швидкого результату)
* **Few-shot** (Шаблони з прикладами "вхід-вихід" для суворого дотримання синтаксису)
* **Chain-of-Thought** (Покрокове міркування моделі для складного рефакторингу та тестів)
* **Role-based** (Призначення експертної ролі для проведення глибокого безпекового аудиту)
